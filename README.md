# 谷粒商城



## 启动配置



1. 数据库

   创建数据库后，运行所有的数据库文件。

2. 服务器

   服务器中需要安装myslq、redis和nacos，推荐使用docker安装

3. 服务器地址

   商城的服务器地址使用虚拟机。配置网桥后更换nacos的地址。

4. OSS地址

   在renrne-fast-vue中src/components/upload中两个vue文件更换为自己的OSS

   除此之外，得在gulimall-third-party模块中的config配置OSS相关配置。
